Версия: 1.0

IP = "127.0.0.1"
PORT = 7777

1. Включение сервера, после чего ожидается подключение клиента.

2. Регистрация или вход пользователя с логиным (уникальный) и паролем.

3. Показ возможных операций.
	3.1 Показать все песни
	3.2 Добавить свою песню
	3.3 Показать свои песни
	3.4 Показать все доступные тэги (добавлять тэги могут только администраторы)
	3.6 Показать купленные песни
	3.5 Купить песню (ввести имя песни и имя автора)
